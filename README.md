# Calculator

Laskin sovellus toiminnallisuudet jotka toimii:

- jako
- kerto
- miinus / plus
- kertoma / jakojäännös
  ei kuitenkaan osaa käsitellä enempää kuin 2 numeroa ja 1 operaattoria syöte tulee siis olla muotoa (numero operaattori numero)

Todos:

- ulkoasun päivitys.
- usean operaattorin käsittely
